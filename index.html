  • Basic Instructions

    1. Splash Screen with music The user is greeted with an entry screen as shown below.
    2. AGLIO displays an entry screen to allow a user to enter the required information to be updated to the real time firebase database. The user can enter his/her details in the app if he/she has left over food. In this page, the user will enter all his/her data and will be updated in the real time firebase database.
    3. -
      • The user is given all the necessary instructions to allow their submission to be updated in the firebase database and to the home screen too.
      • -
      • A sample of the user entry is also given to guide the user to enter accurate information.
      • -
      • If the user enters incorrect information , he/she can clear the text in the fields and write over a new text.
      • -
      • The user can upload a photo taken by a camera and choose from the gallery.
      • -
      • A submit button is provided so that the data can be uploaded to real time firebase database and the main content screen can be updated for all users.

  • FAQ (Frequently Asked Questions)

  • What is AGLIO ?

    AGLIO aims to reduce food wastage and enables users to redistribute unsold/excess food to others in Singapore. AGLIO connects neighbours with each other and with local businesses so left-over food can be shared easily anywhere in Singapore. This could be left-over food from festivals, birthday parties or even from restaurants who are willing to donate their food to less the fortunate. Food waste accounts for about 10 per cent of the total waste generated in Singapore, but only 16 per cent of the food waste is recycled. The rest of it is disposed of at the waste-to-energy (WTE) plants for incineration.
